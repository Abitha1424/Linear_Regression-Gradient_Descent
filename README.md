# 🚀 Salary Prediction using Linear Regression

## 📌 Overview
This project is a simple Machine Learning model built while learning the basics of ML.  
The goal of this project is to predict the salary of a person based on their years of experience using **Linear Regression**.

This helped me understand how supervised learning works in real-world problems.

---

## 🧠 Concepts Learned
- Supervised Learning
- Linear Regression
- Train-Test Split
- Model Training using Scikit-Learn
- R2 Score and Mean Squared Error
- Data Visualization using Matplotlib

---

## ⚙️ Steps in the Project

1. Created a small dataset using Pandas  
2. Selected input (Experience) and output (Salary)  
3. Split data into training and testing sets  
4. Trained model using LinearRegression  
5. Evaluated model using R2 Score  
6. Predicted salary for new experience values  
7. Visualized result using graph  

---

## 💻 Requirements

Install required libraries:

```bash
pip install pandas matplotlib scikit-learn
```

Run the program:

```bash
python salary_model.py
```
